# Marketing Funnel & Conversion Performance Dashboard

## 📊 Project Overview
This project analyzes the complete customer journey from website visitors to final purchases using a Marketing Funnel Dashboard. It helps understand campaign performance, conversion efficiency, and revenue generation through data-driven insights.

## 🎯 Objectives
- Analyze full marketing funnel performance  
- Track user journey from visitors to purchases  
- Identify drop-off stages in the funnel  
- Compare campaign performance  
- Improve conversion rate and ROI  

## 📌 Key Metrics
- Total Visitors: 41,000  
- Total Leads: 11,350  
- Total Purchases: 3,260  
- Total Revenue: 475,000  
- Overall Conversion Rate: 7.95%  

## 📈 Dashboard Insights
- Top-performing campaigns generated higher purchases and revenue  
- Funnel analysis shows customer drop-offs at different stages  
- Conversion rates vary across campaigns  
- Some campaigns delivered stronger ROI than others  
- Revenue exceeded spend in profitable campaigns  

## 📊 Visualizations Used
- KPI Cards  
- Bar Chart (Purchases by Campaign)  
- Column Chart (Revenue by Campaign)  
- Funnel Chart (Marketing Funnel)  
- Comparison Chart (Spend vs Revenue)  
- Conversion Rate Analysis
